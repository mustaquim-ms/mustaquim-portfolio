## 💻 About This Portfolio Repository

This repository houses my digital portfolio. Just as I believe in lightweight, secure, and high-performance infrastructure, I engineered this website from the ground up without heavy frameworks.

### Key Features of this Codebase:

- **Ultra-Premium "Dark Matter" Aesthetic:** Custom CSS variables, dynamic glassmorphism (`backdrop-filter`), and CSS-drawn ambient lighting.
- **Zero-Dependency Animations:** Interactive telemetry nodes, sticky project stacking, and SVG circular progress bars built entirely with Vanilla JavaScript and CSS3.
- **"System OS" Experience:** Features a fully draggable terminal overlay and simulated active-defense event logs to mimic a real Network Operations Center (NOC).
- **CTF Easter Egg:** The site contains a hidden Capture The Flag element for fellow engineers to discover via the UI terminal.
